Task 7: Basic Sales Summary using SQLite and Python
Overview

This project demonstrates how to integrate SQL within Python to extract and summarize sales data from a SQLite database. The task focuses on performing basic data aggregation (total quantity sold and total revenue) and visualizing the results using a bar chart.

The implementation showcases the practical use of SQL queries inside a Python workflow using sqlite3, pandas, and matplotlib.

Objective

Connect Python to a SQLite database

Create and populate a sales table

Execute SQL aggregation queries

Calculate total quantity and revenue per product

Visualize results using a bar chart

Technologies Used

Python 3

SQLite (via sqlite3 module)

Pandas

Matplotlib

Google Colab

Database Structure
Database File

sales_data.db

Table: sales
Column	Data Type	Description
product	TEXT	Name of the product
quantity	INTEGER	Number of units sold
price	REAL	Price per unit
SQL Query Used
SELECT 
    product,
    SUM(quantity) AS total_quantity,
    SUM(quantity * price) AS revenue
FROM sales
GROUP BY product;
Explanation

SUM(quantity) calculates total units sold per product.

SUM(quantity * price) calculates total revenue.

GROUP BY product aggregates records by product name.

Output

The program produces:

A summarized sales table displayed in the console.

A bar chart representing revenue generated by each product.

The visualization provides a clear comparison of product performance based on revenue.

Key Concepts Demonstrated

Database connection using sqlite3.connect()

Table creation using SQL DDL statements

Data insertion using parameterized queries

Aggregation using SUM() and GROUP BY

Loading SQL results into Pandas DataFrame

Data visualization using Matplotlib
